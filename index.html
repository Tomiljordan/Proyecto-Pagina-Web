Tecnologías basicas del desarrollo web

En esta página listaremos de manera breve las tres tecnologías básicas de la web.

¿Cuáles son estas tecnologías?

HTML

Es un lenguaje de marcado donde marcamos qué elementos existen dentro de una página utilizando etiquetas

CSS

En un lenguaje de estilos que nos permite elegir cómo se ven y donde se ubican los elementos de nuestra página.

Javascript

Es el lenguaje de programación en el cual escribimos la interactividad de nuestro sitio web

¿Por qué querríamos aprender estas tecnologías?

Aprender desarrollo web nos beneficiará de distintas maneras:

  • Podremos buscar trabajo en un rubro que no para de crecer.
  • Nos habilita a trabajar para empresas de todo el mundo o a ser independientes.
  • Es un conjunto de tecnologías no muy complicadas que nos introducen al mundo de la programación.
